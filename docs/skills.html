🎯 Skills System

Skills are stored in ~/.miri/skills/ and allow for dynamic extension of the agent's capabilities.

Skill Formats

  • Single-file (.md): A plain Markdown file with YAML frontmatter. Recommended for most skills.
  • Directory-based: A folder containing a SKILL.md file. Used for skills with bundled scripts or assets.
  • Hybrid: A .md file with a matching directory for a scripts/ subdirectory.
---
name: Web Analysis
description: Advanced web page analysis instructions.
version: 1.2.0
tags: [web, scraper]
---
# Instructions
When performing web analysis...

Core Skills & Auto-Activation

Miri bootstraps core skills from its templates. These are automatically activated in the primary session (miri:main:agent):

  • learn: The primary skill for discovery and installation via agentskill.sh.
  • skill_creator: Helps the agent design and write new skills for itself.

Script Inference

Miri can automatically turn scripts into agent tools. Supported extensions: .sh, .py, .js.

  • Global Scripts: Located in scripts/ in the storage folder.
  • Skill Scripts: Located in a skill's scripts/ subdirectory (e.g., ~/.miri/skills/my-skill/scripts/).

Management Commands

Action Tool / Command
Install/Search /learn <query>
Activate skill_use <name>
Uninstall skill_remove <name>
List Available Admin API: GET /api/admin/v1/skills
