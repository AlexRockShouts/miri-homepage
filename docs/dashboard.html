📊 Miri Dashboard

A modern, high-performance web dashboard built with SvelteKit 5 and Tailwind CSS 4 for interacting with and managing the Miri Autonomous Agent.

Real-time Reactive Interface

The dashboard leverages low-latency WebSockets and the @miri/sdk to provide a seamless, real-time experience for monitoring and interacting with the agent.

Core Sections

1. Admin Dashboard (Overview)

The primary control center for the Miri ecosystem. It provides a comprehensive view of the server's current state:

  • Server Health: Real-time monitoring of server status, uptime, and resource usage.
  • Active Sessions: View and manage all ongoing chat sessions, including specialized task sessions.
  • Human Profiles: Inspect and manage the stored knowledge about users (Human LTM).
  • System Configuration: Visual overview of the active config.yaml settings.

2. Reactive Prompt (Chat)

A sophisticated chat interface designed for direct interaction with the Miri agent:

  • Streaming Responses: High-speed SSE/WebSocket streaming for real-time thought markers and agent output.
  • Thought Visualization: View the agent's internal reasoning process (Mole-Syn traces) as they happen.
  • Tool Execution Logs: Real-time feedback on tool calls, including file operations and web searches.
  • Multi-Session Support: Switch between different sessions (e.g., miri:main:agent) with ease.

3. The Brain (Memory Viewer)

A dedicated interface for visualizing and managing Miri's long-term memory:

  • Facts Explorer: Search and filter the atomic facts extracted by the maintenance loop.
  • Summaries View: Review the narrative context summaries created from past interactions.
  • Topology Analysis: Inspect the stability scores and bond distributions (D, R, E) of the agent's reasoning.
  • Priority Sorting: Sort memories by confidence, access count, or topology source to understand what the agent values most.

4. Skills Management

Manage and discover new capabilities for the agent:

  • Installed Skills: List and inspect all locally available skills and inferred scripts.
  • Command Reference: A real-time list of all available agent tools and their descriptions.

Technical Stack

Layer Technology
Framework SvelteKit 5 (SSR + Hydration)
Styling Tailwind CSS 4 + shadcn-svelte
Communication WebSockets + Axios (@miri/sdk)
Icons Lucide Svelte

Setup & Configuration

The dashboard is configured via environment variables in a .env file. For detailed setup instructions, including local SDK linking, see the Getting Started guide.
