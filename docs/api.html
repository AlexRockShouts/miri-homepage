📡 API & SDK

Miri provides a comprehensive REST and WebSocket API for interaction and management.

Authentication

  • Server Key: Required for /api/v1/* and /ws. Pass via X-Server-Key header or WS sub-protocol.
  • Admin Auth: Basic Auth (configured in config.yaml) for /api/admin/v1/*.

Standard API (v1)

Requires X-Server-Key authentication. All endpoints return JSON unless specified otherwise.

POST /api/v1/prompt

Blocking prompt execution. Returns the final agent response after all reasoning and tool calls are complete.

Request Body

{
  "prompt": "string",
  "session_id": "string (optional)",
  "stream": false
}

Response

{
  "response": "string",
  "usage": {
    "prompt_tokens": 120,
    "completion_tokens": 45,
    "total_tokens": 165
  }
}

GET /api/v1/prompt/stream

Real-time streaming via Server-Sent Events (SSE). Streams thoughts, tool calls, and final responses as they happen.

  • [Thought: node_name started] - Engine entering a graph node.
  • [Tools: tool_name finished] - Tool execution results.
  • [Usage: ...] - Final token usage and cost.

WebSocket (/ws)

Full-duplex real-time communication. Supports the miri-key sub-protocol for authentication.

Events

  • Inbound: {"prompt": "hi", "stream": true}
  • Outbound (Chat): {"response": "...", "source": "agent"}
  • Outbound (Task): {"response": "...", "source": "task", "task_name": "..."}

Admin API (v1)

Requires Basic Authentication. Manage server state and monitor the cognitive engine.

Endpoint Method Description
/admin/v1/health GET Server status and component health.
/admin/v1/config GET/POST View or update server and model configuration.
/admin/v1/brain/topology GET Latest Mole-Syn stability score and distribution.
/admin/v1/brain/facts GET List all extracted facts from long-term memory.
/admin/v1/sessions GET List active chat sessions and their stats.
/admin/v1/skills GET List all installed skills and inferred script tools.
/admin/v1/tasks GET List and monitor scheduled cron tasks.
/admin/v1/human GET/POST Manage stored human profiles and context.
